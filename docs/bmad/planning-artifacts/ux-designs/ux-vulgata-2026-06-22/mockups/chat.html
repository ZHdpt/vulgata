风险评估的完整流程是什么？
风险评估流程涉及三个系统协同工作：

① 手机银行接收用户发起的贷款或转账请求
② 风控引擎接收请求，根据预设规则评估该交易的风险等级（低/中/高）
③ 中低风险交易交由核心银行系统完成资金操作；高风险交易触发人工审核

整个过程中还会调用反欺诈系统和征信接口进行辅助判断。
📄 引用：风控引擎业务逻辑文档 · 核心银行审批流程 · 反欺诈规则清单
如果客户信用评级是 C 级呢？
C 级客户会触发更严格的校验流程：

① 风控引擎会将风险等级自动上调一级（低→中，中→高）
② 增加了收入验证环节，需提交近三个月银行流水
③ 高风险交易即便金额较小也需要支行经理审批
